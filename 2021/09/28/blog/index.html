搭建个人博客前的准备

官网下载：

git node.js

git和node.js的安装一直next就行，期间不需要选择任何项目。git我是直接安装到\F:Git\目录下，nodejs在\F:nodejs\目录下。

ps:个人建议不要去改动系统变量里的路径，不然后面安装hexo时可能会出现一些问题（各自电脑不同）

测试是否安装成功

打开安装好的git bash here或者在cmd中测试

如果都显示版本号，说明安装成功

安装hexo

在nodejs文件夹右键git bash here

安装好之后测试一下，出现版本号证明安装成功

因为我之前已经安装过，所以比较快，如果你安装的时候一直不动，耐心等待。多试几次，我安装时就是多试几次后才安装成功的。

创建博客

创建一个空文件夹，我在F盘创建了一个名为blog的文件夹，在文件夹空白处右键选择git bash here

记住一定要是空白文件夹，而且最好是一级目录。hexo init很小气

出现**INFO Start blogging with Hexo!**说明创建成功

执行hexo init时，可能会一直停留在INFO Install dependencies这一步,不要着急耐心等待，可能还会出现连接github失败，多试几次hexo init(本人花了一早上才连接成功)
